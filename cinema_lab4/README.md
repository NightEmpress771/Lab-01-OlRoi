# Лабораторна робота №4 — База даних кінотеатру

## Опис
Проєкт реалізує базу даних кінотеатру з використанням PostgreSQL і веб-інтерфейсу Flask.

## Структура
- `console_app/` — консольна частина
- `web_app/` — веб-застосунок Flask
- SQL-файли — створення структури, даних, представлень, тригерів
- `report_lab4_cinema.docx` — звіт до лабораторної
